Install¶

From Source¶

Clone the repository in the appropriate GOPATH subdirectory:

$ mkdir -p $GOPATH/src/github.com/Fantom-foundation/
$ cd $GOPATH/src/github.com/Fantom-foundation
[...]/Fantom-foundation$ git clone https://github.com/Fantom-foundation/go-lachesis.git

The easiest way to build binaries is to do so in a hermetic Docker container. Use this simple command:

[...]/lachesis$ make dist

This will launch the build in a Docker container and write all the artifacts in the build/ folder.

[...]/lachesis$ tree build

Go Devs¶

Lachesis is written in Golang. Hence, the first step is to install Go version 1.9 or above which is both the programming language and a CLI tool for managing Go code. Go is very opinionated and will require you to define a workspace where all your go code will reside.

Dependencies¶

Lachesis uses Glide to manage dependencies. For Ubuntu users:

[...]/lachesis$ curl https://glide.sh/get | sh
[...]/lachesis$ glide install

This will download all dependencies and put them in the vendor folder.
